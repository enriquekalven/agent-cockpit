The `PIIScrubber` is an automated middleware that sits between the user and the LLM.

## ğŸ§ª Evidence Packing (explainability)
Every response generated by the Optimized Engine can be wrapped in an **Evidence Packet**.
- **The Packet**: A structured JSON object containing:
    - **Reasoning Path**: The thought chain the agent followed.
    - **Sources**: Direct links/snippets to the RAG documents used.
    - **Tool Evidence**: The raw output of function calls.
- **The Benefit**: Users can click "Show Sources" in the **Face (A2UI)** to see exactly why the agent gave a specific answer.
